## 🔧 Troubleshooting

**Q: The app won't start**
- Make sure you're running the `.exe` file directly
- Check Windows Defender didn't block it (right-click → Properties → Unblock)

**Q: PDF won't open**
- Make sure it's a valid PDF file
- Try a different PDF to test
- Check file isn't corrupted

**Q: Highlights not saving**
- Check you have write permissions to AppData folder
- Try running as administrator once

**Q: App is slow**
- Large PDFs may take time to load
- Close other applications to free up memory
- Reduce zoom level if page rendering is slow
